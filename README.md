# Simulador Camada Física - TR1 2022/1

## Dependências
- g++
- GNU make

## Como compilar e rodar o simulador

### Modo debug
```
prompt> make
prompt> bin/simulador
```

### Modo release
```
prompt> make release
prompt> bin/simulador
```
