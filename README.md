# Simulador Camada Física - TR1 2022/1

## Dependências
- g++
- GNU make
- SDL2

### Instalando dependências no Ubuntu
```
apt-get install libsdl2-dev g++ make
```

## Como compilar e rodar o simulador

### Modo debug
```
prompt> make
prompt> bin/simulador
```

### Modo release
```
prompt> make release
prompt> bin/simulador
```

## Gerando o arquivo `zip` de submissão
```
prompt> make submit
```
